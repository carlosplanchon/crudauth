## Reporting issues

- **Bugs and features:** open an issue on [GitHub](https://github.com/benavlabs/crudauth/issues).
- **Security vulnerabilities:** do **not** open a public issue. See the
  [security policy](https://github.com/benavlabs/crudauth/blob/main/SECURITY.md) (email or a
  private GitHub advisory).
